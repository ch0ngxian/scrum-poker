## Getting Started
1. Clone this repo by `git clone git@github.com:ch0ngxian/scrum-poker.git`
2. Open the repo by `cd scrum-poker`
3. Copy `.env.example` to `.env.local` and fill in the environment variables
4. Install packages by `npm i`
5. Start the application by `npm run dev`
